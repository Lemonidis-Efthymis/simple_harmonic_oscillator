# Simple Harmonic Oscillator Simulation

A Python simulation of a **simple harmonic oscillator (mass-spring system)** using NumPy and Matplotlib.

The script calculates and plots:
- Mass displacement over time
- Cosine-based oscillatory motion

## üìê Physics Model

- Mass: 0.5 kg  
- Spring constant: 20 N/m  
- Initial displacement: 0.2 m  
- Total simulation time: 5 s  

The motion follows:  
\[
x(t) = x_0 \cos(\omega t), \quad \omega = \sqrt{\frac{k}{m}}
\]
